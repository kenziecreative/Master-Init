   ```json
   "SessionStart": [
     {
       "matcher": "startup",
       "hooks": [{ "type": "command", "command": "bash \"$CLAUDE_PROJECT_DIR/.claude/hooks/session-start.sh\" startup", "timeout": 10 }]
     },
     {
       "matcher": "resume",
       "hooks": [{ "type": "command", "command": "bash \"$CLAUDE_PROJECT_DIR/.claude/hooks/session-start.sh\" resume", "timeout": 10 }]
     },
     {
       "matcher": "clear",
       "hooks": [{ "type": "command", "command": "bash \"$CLAUDE_PROJECT_DIR/.claude/hooks/session-start.sh\" clear", "timeout": 10 }]
     },
     {
       "matcher": "compact",
       "hooks": [{ "type": "command", "command": "bash \"$CLAUDE_PROJECT_DIR/.claude/hooks/session-start.sh\" compact", "timeout": 10 }]
     }
   ]
   ```
   Note: The compact matcher IS registered — the script itself handles the no-op. This ensures the hook chain is complete even if behavior changes later.

           "command": "bash \"$CLAUDE_PROJECT_DIR/.claude/hooks/session-end.sh\"",

     }
   ]
